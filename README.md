## パッケージ構成
- `com.github.arknote02` ベース（グループID）
  - `account` アプリ名（アーティファクトID）
    - `domain` ドメイン
    - `infrastructure` インフラ  
      フレームワークや外部リソースに依存するコード。  
      エンドポイントは別パッケージに切り出してもいいかも。
    - `usecase` ユースケース  
      インタフェースのみにしてフレームワークへの依存を排除。
